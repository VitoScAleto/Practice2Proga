Множество – совокупность объектов, или элементов a принадлежащих A.

Алгебра – непустое множество A с набором операций <f1, f2…f3>. 
Алгебра: <A: f1, f2 … f3> множество замкнутое на своих операциях. 
В каждой алгебре есть набор чисел(символов), операций и выдел.элемента.

Носитель алгебры – непустое множество A.

Тип алгебры – не пустое подмножество множества A^n наз. n-местным 
отношением на множестве A.

Сигнатура алгебры – <f1, f2…f3> набор отображений f = A^n → A наз. n-местной 
операцией на множестве A.

Группа – G(алгебра) = <G: x, ^-1, e> с типом алгебры (2, 1, 0) с r-ностями наз. группой, 
если 1) для любых a, b, c из G, где a X (b X c) = (a X b) X c (декартово произведение); 
2) для каждого a из G, где a X a^(-1) = a^(-1) X a = e; 
3) для каждого a из G, где a X e = e X a = a.

Кольцо – < k: +, -, *, 0 > с типом алгебры <2, 1, 2, 0>, если 
1) k = <k; +, -> абилеева группа (число1*число2 = число3*число4); 
2) для каждого x из k, где x*0 = 0*x = 0; 
3) для каждого x, y, z из k, где (x+y)*z = x*z+y*z = z(x+y)

Ассоциативное кольцо – для каждого x, y, z, где (xy)z = x(yz)

Кольцо с единицей – если существует 1, такая что x*1 = 1*x = x.

Коммутативное кольцо – есть для x, y из k выполняется условие: xy = yx.

Поле – < k: +, -, ^(-1), *, 0, 1 > с типом алгебры <2, 1, 1, 2, 0, 0>, если 
1) k = k\{0} абилеева группа (число1*число2 = число3*число4); 
2) выполнено тождество: x(y+z)=xy+xz.

Делимость в кольце целых чисел (факт делимости, математическая запись) – делимость означает, 
что существует некоторое ь, такой что a = b*ь.  2|6 – математическая запись, 
5(перечеркнуть)|6. Если a|b и a|c, то a|(bЪс), где Ъ – некоторая операция (+, -, *), 
обратно тоже работаем.
